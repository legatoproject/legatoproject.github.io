AirVantage Control

This sample demonstrates an LWM2M Control app communicating with AirVantage. It has an .adef file with executables, processes, and bindings sections, and @ .cdef file with requires and sources sections.

Run this to install the sample app: 

$ cd apps/sample/lwm2mAirVantageControl
$ make wp85
$ instapp lwm2mAirVantageControl.wp85.update <ip address>

Start App
