legato

Use the legato tool to run the Legato framework.

Usage

legato [start | stop | restart | version | help]

legato start

Starts the Legato framework.

legato stop

Stops the Legato framework.

legato restart

Restarts the Legato framework.

legato version

Displays the intstalled Legato framework version.

legato help

Displays usage help.
