NAME

releaselegato - release legato project

SYNOPSIS

releaselegato [OPTIONS]

DESCRIPTION

Once Legato has been built for selected targets, this script stages files and produces a tarball.

Options:

  • -t [TARGETS]: comma-separated list of targets the package should be generated for (eg 'ar7,wp85')
